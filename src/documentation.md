## `RemoveComments(text: string): string`

### Назначение: Удаляет однострочные и многострочные комментарии из исходого файла
### Параметры: text (string): Исходный текст файла, из которого нужно удалить комментарии
### Возвращаемое значение:  string: Новый текст, в котором все комментарии удалены.
### Пример: 
было:
```cpp
#include <iostream>
// комментарий
int main() {
    /* многострочный
    комментарий */
    std::cout << "Hello \"World\""; // комментарий
    return 0;
}
```
стало:
```cpp
#include <iostream>

int main() {
    
    std::cout << "Hello \"World\""; 
    return 0;
}
```

## `activate(context: vscode.ExtensionContext): void`
### Назначение: Функция активации расширения. Выполняется при активации плагина VS Code
### Параметры: context (vscode.ExtensionContext): Контекст расширения, предоставляемый VS Code
### Возвращаемое значение: void: Функция не возвращает значения

## `deactivate(): void`
### Назначение: Функция деактивации расширения. Выполняется при выгрузке (деактивации) плагина VS Code
### Параметры: нет
### Возвращаемое значение:void: Функция не возвращает значения
